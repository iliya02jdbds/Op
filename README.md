# Neon Arena — راهنمای راه‌اندازی

بازی سه‌بعدی PvP آنلاین **چندنفره** (۲ تا ۸ بازیکن، فری‌فور-آل) با تفنگ، حرکت آزاد، دوربین سوم‌شخص، تم دارک/نئون، گرافیک Three.js و همگام‌سازی زنده با Socket.io.
چون بازی آنلاینه، به یه **سرور در حال اجرا** نیاز داره (نه فقط یه فایل استاتیک).

## ساختار پروژه
```
sword3d/
  server.js       ← سرور Node.js (matchmaking + همگام‌سازی + امتیازها)
  package.json
  public/
    index.html    ← کل بازی (گرافیک سه‌بعدی + منطق کلاینت)
```

## مرحله ۱: هاست رایگان روی Render.com

1. یه اکانت رایگان تو https://render.com بساز.
2. این پوشه (`sword3d`) رو به یه ریپازیتوری GitHub آپلود کن.
3. تو Render: **New → Web Service** → ریپازیتوری‌ت رو انتخاب کن.
4. تنظیمات:
   - **Build Command:** `npm install`
   - **Start Command:** `npm start`
   - **Instance Type:** Free
5. بعد از دیپلوی، یه آدرس مثل `https://sword-duel-3d.onrender.com` بهت میده.

> نکته: پلن رایگان Render بعد از چند دقیقه بی‌کاری "می‌خوابه" و اولین درخواست بعدی چند ثانیه طول می‌کشه تا بیدار شه. برای بازی رایگان طبیعیه.

جایگزین‌ها: **Railway.app** یا **Fly.io** هم پلن رایگان/ارزان دارن و مراحل مشابهی دارن.

## مرحله ۲: تست محلی (اختیاری، قبل از دیپلوی)
اگه Node.js رو روی سیستم خودت نصب داری:
```bash
cd sword3d
npm install
npm start
```
بعد `http://localhost:3000` رو تو مرورگر باز کن. اگه از دو تب مرورگر باز کنی، می‌تونی با خودت دوئل کنی و تست کنی.

## مرحله ۳: وصل کردن به بات تلگرام

1. برو تو BotFather → ربات‌ت → **Games** → `/newgame`
2. آدرس همون سروری که از Render گرفتی رو (مثلاً `https://sword-duel-3d.onrender.com`) به‌عنوان URL بازی بده.
3. تو کد ربات‌ت (هرجا که پیام می‌فرستی)، با متد API زیر بازی رو تو چت بفرست:
   ```
   sendGame(chat_id, game_short_name)
   ```
4. وقتی کاربر روی دکمه‌ی «Play» بازی می‌زنه، تلگرام همین صفحه رو تو WebView باز می‌کنه؛ بازیکن اسمش رو می‌زنه، رو دکمه‌ی «پیدا کردن حریف» می‌زنه، و منتظر یه کاربر دیگه از بات می‌مونه که همزمان وارد بازی بشه.

## محدودیت‌های نسخه‌ی فعلی (صادقانه بگم)
- **گرافیک استایلایز نئون/دارک، نه مدل‌های واقعی انیمه**: چون این محیط امکان ساخت مدل‌های سه‌بعدی پیچیده یا ایمپورت اسپرایت انیمه رو نداره، کاراکترها لوپلی (استوانه+کره) با رنگ‌های نئون درخشان و افکت‌های glow هستن؛ اگه مدل/تکسچر مشخصی داری که می‌خوای استفاده بشه، بفرستش تا اضافه‌ش کنم.
- **بدون سیستم اکانت واقعی**: بازیکن‌ها فقط با یه اسم موقت شناخته میشن.
- **آنتی‌چیت محدود**: موقعیت بازیکن سمت کلاینته و فقط اصابت گلوله سمت سرور تأیید میشه؛ برای بازی دوستانه کافیه ولی در برابر کاربر مخرب صد در صد امن نیست.
- **جفت‌سازی چندنفره ساده**: وقتی حداقل ۲ نفر منتظرن، یه پنجره‌ی ۶ ثانیه‌ای باز میشه که بقیه (تا ۸ نفر) هم بتونن بپیوندن؛ رنکینگ/لیگ نداره.
- پلن رایگان هاست معمولاً همزمان محدودیت تعداد کاربر داره؛ برای استفاده‌ی جدی‌تر باید پلن پولی بگیری.

اگه بخوای می‌تونم بعداً ری‌لود اسلحه، نقشه‌های بیشتر، مینی‌مپ، یا آنتی‌چیت قوی‌تر سمت سرور هم بهش اضافه کنم.
